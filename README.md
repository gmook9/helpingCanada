# HelpingCanada
### Connecting donors with verified, trustworthy charities in Vancouver.

HelpingCanada is a transparent, referral-free directory that helps people find legitimate charitable organizations across Vancouver. No partnerships & no incentives, just clear information to support informed giving.

## Mission
Make it easier for people to help people by guiding donors toward reputable, community-focused nonprofits.

## Features
- Verified Vancouver charities  
- No referral fees or sponsorships  
- Clear, unbiased information  
- Privacy-first (no data collection)
